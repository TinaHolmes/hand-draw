# hand-draw
隔空手势绘图系统  这是一个基于 MediaPipe Hands 和 Canvas API 构建的实时网页端交互系统。它利用人工智能视觉算法，让用户无需触碰屏幕，仅通过摄像头捕捉手势即可在空中进行绘画创作。     Air-Drawing System  An interactive real-time web-based system built with MediaPipe Hands and Canvas API. Using AI computer vision, users can draw in the air via their webcam without any physical contact with the screen.
